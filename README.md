# Boss AI: Fast Python Voice Assistant 🚀

Boss AI is a high-performance **Virtual Assistant** built with **Python**, **Groq Llama 3**, and **Flet UI**. It features a modern voice chat interface and lightning-fast response times.

## 🌟 Features
- **Voice Recognition:** Powered by Google Speech API.
- **Fast Responses:** Uses Groq's Llama-3.3-70b model.
- **Modern UI:** Built with Flet for a smooth, app-like experience.
- **Automation:** Can open apps and play music on YouTube.

## 📥 How to Download and Setup
1. **Clone the repo:** `git clone https://github.com/mrstanger32-a11y/boss-ai.git`
2. **Install requirements:** `pip install -r requirements.txt`
3. **Setup API Key:** Create a `.env` file and add `GROQ_API_KEY=your_key_here`.
4. **Run:** `python main.py`

## 🛠️ Technologies Used
Python, Flet, Groq API, Edge-TTS, SpeechRecognition.
